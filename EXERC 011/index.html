Imagens dinâmicas

Abra o navegador em celulares, tablets ou computadortes e a imagem se adaptada dinamicamente ao tamanhoda tela.

para isso basta usar a tag < PICTURE >

<picture> <source media="(max-width:750px)" srcset="images/foto-p.png" type="image/png"> <source media="(max-width:1050px)" srcset="images/foto-m.png" type="image/png"> <img src="images/foto-g.png" alt="imagens flexíveis"> <picture>

Inserindo arquivos de áudio

Inserindo áudio com a tag < AUDIO src=" " controls autoplay >

<audio src="media/audio_1.m4a" controls autoplay>

Inserindo com a tag < AUDIO > < SOURCE >

<audio preload="metadata" autoplay controls loop>

Inserindo arquivos de vídeos

Inserindo vídeos hospedados localmente.

Inserindo áudio com a tag < VIDEO src=" " controls poster >

<video src="media/video_1.mov" poster="images/thumb.png" width="300px" controls>

Inserindo com a tag < VIDEO > < SOURCE >

<video width="300px" poster="images/thumb.png" controls> <source src="media/video_1.mov" type="video/mp4">
