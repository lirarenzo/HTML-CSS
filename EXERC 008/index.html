Principais formtações

Negrito

Nessa frase temos um texto em negrito usando a tag < b > (não semântica).

Nesta outra frase temos um texto em destaque usando a tag < STRONG > (semântica).

Itálico / ênfase

Nesta frase temos um texto em itálico usando a tag < i > (não semântica).

Nesta frase temos um texto em ênfase usando a tag < EM > (semântica).

Usar sempre as tags STRONG e EM para negrito e ênfase (itálico)

Texto marcado

Inserindo um texto com marcação usando a tag MARK Style (usar o Style para configurar a cor de fundo da marcação)
